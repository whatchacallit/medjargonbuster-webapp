# SumMed - Web Application

This is the web application frontend for the https://summed.org project.

Tech stack:
- Typescript, 
- npm 
- craco
- React (with hooks)
- ANT Design components 

# How to get started

- clone this repo ;-)
- npm install
- npm run start
- go to http://localhost:3000
- ... you know what to do :)
